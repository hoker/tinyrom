# Security Policy

## Scope

tinyrom is calibration tooling. It manipulates ROM bytes and definition files, but it does not validate that a proposed tune is safe for any specific engine.

## Reporting

Please report security issues privately to the maintainers rather than opening a public issue.

## Expectations

- Do not submit proprietary ROMs, XDFs, passwords, or vendor installers in issues or pull requests.
- Treat all generated calibration artifacts as sensitive until reviewed.
- Verify any change before flashing hardware.

## Safety Note

tinyrom is provided for research, education, and tooling. Users are responsible for validating the correctness and safety of calibration changes before applying them to real vehicles or ECUs.
